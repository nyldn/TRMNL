# Daily Quote

Minimal static TRMNL Recipe example.

This example keeps all data in `src/settings.yml` as `static_data`, making it useful for local preview and structure testing.

The export script emits the five import-documented files: `settings.yml` plus four layout templates.

## Commands

```sh
../../../scripts/validate-recipe.sh .
../../../scripts/preview-recipe.sh .
../../../scripts/export-recipe.sh .
```
